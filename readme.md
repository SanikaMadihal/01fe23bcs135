## üíª Algorithmic Implementations

This repository contains a collection of essential data structures and algorithms implemented during the course, which serve as the building blocks for the **Smart City Planning** project.

| Category | Algorithms Implemented | Complexity | Application in Smart City |
| :--- | :--- | :--- | :--- |
| **Graph Theory** | Dijkstra's, Kruskal's, Prim's | $O(E \log V)$ | Routing & Utility Networking |
| **Sorting & Searching** | Merge Sort, Quick Sort | $O(n \log n)$ | Sorting Census & Resource Data |
| **Dynamic Programming** | Floyd-Warshall, Knapsack | $O(V^3)$ / $O(nW)$ | All-pairs Routing & Budgeting |
| **Advanced Trees** | Segment Trees, Fenwick Trees | $O(\log n)$ | Range Queries for Population |
| **Search Techniques** | BFS, DFS | $O(V + E)$ | Network Traversal & Connectivity |
